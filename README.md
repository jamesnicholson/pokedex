# An Pokedex App
A small pokedex of the first 152 pokemon.
 `yarn start` to get it going

Demo URL: http://161.35.115.185:3300/

## tech used
- TypeScript
- React
- styled-components
- docker
- pokeapi.co

## This project runs on docker
`docker-compose up -d  --build`
